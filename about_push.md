# MANUAL for GIT

## **Commands in GIT**

1. "git config --global user.name "Имя пользователя" - задание имени пользователя;

2. "git config --global user.email "Адрес электронной почты" - задание почты пользователя;

3. "git init" - Инициализация репозитория;

4. "git add "Название файла" - добавление файла (для автозаполнения имени файла введите несколько первых букв названия и нажмите *Tab*);

5. "git status" - проверка статуса репозитория;

6. "git commit -m "Комментарий" - сохранение изменений с комментарием;

7. "git log" - просмотр истории изменений;

8. "git diff" - просмотр изменений между текущим состоянием файла и сохраненным;

9. "git checkout" - возвращение к состоянию файла до "*commit*";

10. "git checkout master" - возвращение к текущему состоянию файла;

## **Сommands for working with branches**

1. "git branch" - вывод списка веток на экран;

2. "git branch <имя ветки>" - создать новую ветку;

3. "git checkout <имя ветки>" - переход на ветку;

4. "git merge <имя ветки>" - слияние веток. Вызываем из той ветки в которой надо добавить;

5. "git branch -d <имя ветки>" - удаление ветки;

6. "git log --graph" - графическое отображение веток ("дерево");

## **Сommands for working with repositories**

1. "git clone" - клонировать удаленный репозиторий на свой ПК;

2. git pull" - "стянуть" все из удаленного репозитория и сделать слияние с нашей версией;

3. "git push" - "толкнуть" (отправить) нашу версию репозитория на удаленный репозиторий;
