# Инструкция по работе с Git

![THE GIT LOGO](gitlogo.png)

Первым делом нужно установить Git на ваш компьютер и после чего начинать изучение

### Глава 1: Создание проекта

1. Для начала нужно прописать команду *git init* 
для создания репозитория
2. После чего можно добавить файлы проекта
командой: *git add название файла с расширением*

  Пример: *git add example.txt*

3. После чего можно написать *git status* что бы узнать что ещё осталось добавить в файлы

 ### Глава 2: Создание commit и переключение между ними

 1. Что бы сделать сохранение нужно сделать commit - сохранение, фиксирование с помощью команды:
 *git commit -m "комментарий в кавычках"*

 (Рекомендуется делать commit после например создания нового функционала, исправления множества ошибок или завершения рабочего дня)

  2. Что бы переключаться между версиями commit нужно сначала увидеть список всех commit по команде;
 git log которая отобразит все commit где будут написаны id(пример e5frg45657 и т.д.) всех сохранений (что бы выйти из режима наджмите клавишу q с включённой английской раскладкой)
 3. После того как мы нашли интересующее нас сохранение, пишем команду 
 git checkout первые 4 символа сохранения
 4. Что бы вернуться к нынешнему сохранению надо написать
 *git checkout master*

  ### Глава 3: Дополнительные функции и сохранение изменений

  1. Что бы сохранить изменения в файле можно прописать команду
  *git add название файла с расширением* или сделать commit с сохранением всех файлов командой
  *git commit -am "название сохранения в кавычках"*
  2. Что бы посмотреть разницу между коммитами можно прописать команду
  *git diff первые 4 символа коммита* (что бы выйти из режима просмотра сравнения нажмите клавишу q английской раскладкой)
 
  ### Глава 4: Работа с своим проектом на github

  1. Для начала работы с github нужно войти в аккаунт в браузере по умолчанию и в visual studio авторизовать аккаунт github в меню аккаунта
  2. После этого нужно написать команду 
  *git remote add название сылка на ваш репозиторий* это действие добавит в git под вашим названием ваш репозиторий
  
    Пример: *git remote add name httрs://github.com/TimurIslamgulov/2982.git*
  
  3. Перед тем как выложить репозиторий рекомендуется прописать команду
  *git branch -M main* и потом уже написать команду для выкладывания
  *git push -u название main*

    Пример: *git push -u name main*

  4. Что бы получить проект с github надо написать команду
  
    *git pull*

  ### Глава 5: Предложение своей идеи на github

  1. Для этого во первых на сайте github нужно создать fork интересующего проекта и после чего копируем её и потом пишем команду
  *git clone сылка вашего форка*
  
  2. После чего нужно переместиться в папку, которая создалась
  *cd путь папки*

    Пример: *cd Forks/FirstFork*

  3. После чего желательно создать новую ветку по мимо основной, переключиться на неё и редактировать изменения уже в ней
  4. Что бы отправить изменения нужно сделать commit ветки и написать команду 
  *git push --set-upstream ваше название репозитория для гита название ветки*

    Пример: *git push --set-upstream name vetka2*

  5. После чего на сайте github можно будет отправить изменения кнопкой "Cpmpare & pull request" и в появишемся написать изменения ветка, откуда репозиторий, название ветки и комментарии
