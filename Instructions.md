Это какое-то проект на github аккаунте.


1. Делаем ФОРК (fork) интересующего нас репозитория.
2. Мы делаем git clone для нашей версии этого репозитория
3. Затем переходим в эту папку через команду **cd имя папки** 
4. Мы создаем ветку с предлогаемыми изменениями. И обязательно, проверить что мы в этой ветки!
5. Производим все изменения только в этой ветке. Сохраняем и делаем commit (написанно что конкретно мы сделали!)
6. Отправляем эти изменения на свой аккаунт (push).
7. В окне на Github появляется возможность отправить pull request
