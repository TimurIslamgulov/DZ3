## Работа с удаленными репозиториями ##

Удалённые репозитории — это модификации проекта, которые хранятся в интернете или ещё где-то в сети. Их может быть несколько, каждый из которых как правило доступен для вас либо только на чтение, либо на чтение и запись. Совместная работа включает в себя управление удалёнными репозиториями и помещение (push) и получение (pull) данных в и из них тогда, когда нужно обменяться результатами работы. (источник - https://linuxcookbook.ru/books/progit/ch02s05.html)

Чтобы просмотреть какие удалённые серверы у вас уже настроены, следует выполнить команду git remote. Она перечисляет список имён-сокращений для всех уже указанных удалённых дескрипторов. 

На практическом задании мы говорили о последовательности дейтсвий при pull request

*Заходим в репозиторий
*Форкаем его
*Клонируем к себе
*Работаем в нем
